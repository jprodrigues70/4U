### Tech

4U utiliza algumas tecnologias de Desenvolvimento WEB:

* [PHP] - Linguagem de Programação usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web.
* [HTML] -  linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.
* [CSS] - linguagem de folhas de estilo utilizada para definir a apresentação de documentos escritos em uma linguagem de marcação, como HTML ou XML. O seu principal benefício é a separação entre o formato e o conteúdo de um documento.
* [SASS] - pré-processadore e framework de folhas de estilo para auxiliar na produtividade de códigos, principalmente no que diz respeito a repetição de uma mesma ação, diversas vezes.
* [jQuery] - biblioteca JavaScript cross-browser desenvolvida para simplificar os scripts client side que interagem com o HTML.
